# UIautomator

## Install

- install the `uiautomator2` package
    - `pip install uiautomator2`
    - my using version is 2.2.0
        - `pip install -Iv uiautomator==2.2.0`

## Exec
- `python3 uiautomator.py config/config.txt outputfile config/w1.txt`

## Reference
